# Thinking Typology: A Learner Profile Framework for Grade 5-6

## Theoretical Foundations

This typology draws from five established frameworks in educational psychology and cognitive science:

| Framework | Author(s) | Key Contribution to Our Design |
|-----------|-----------|-------------------------------|
| **Mental Self-Government** | Sternberg (1997) | Thinking is not about ability level but about *style* — how someone prefers to use their abilities. Our system describes preference, not deficit. |
| **SOLO Taxonomy** | Biggs & Collis (1982) | Learning progresses through observable structural levels: from single ideas (unistructural) to connected ideas (relational) to abstract transfer (extended abstract). We use this to define depth. |
| **Revised Bloom's Taxonomy** | Anderson & Krathwohl (2001) | Cognitive processes form a hierarchy: Remember → Understand → Apply → Analyze → Evaluate → Create. We use this to classify what type of thinking a student's work reveals. |
| **Deep vs Surface Learning** | Marton & Säljö (1976) | Learners approach the same task in qualitatively different ways: surface (reproduce information) vs deep (transform meaning). Our typology captures this spectrum. |
| **Formative Fiction** | Landy (2012) | Literary engagement trains cognitive capacities — the value is in the process, not the content absorbed. Our system measures capacity development, not knowledge accumulation. |

### Key Design Principle from Sternberg

> "An ability refers to how well someone can do something. A style refers to how someone *likes* to do something — a preferred way of using the abilities one has."

Our typology describes **thinking styles**, not ability levels. No type is better or worse — each has strengths and growth areas.

## Part 1: The Three Dimensions

Instead of 8 separate capacity scores, we organize student thinking along **three observable dimensions**. Each is a spectrum, not a binary.

### Dimension 1: Depth of Processing

*Adapted from SOLO Taxonomy + Marton & Säljö's deep/surface distinction*

How deeply does this student engage with material?

```
Surface ◄────────────────────────────► Deep
```

| Level | SOLO Equivalent | What It Looks Like in Student Work |
|-------|----------------|-----------------------------------|
| **Recall** | Pre/Unistructural | Restates facts from the text. Writing lists information without connecting ideas. |
| **Connect** | Multistructural | Identifies multiple relevant pieces of information. Writing presents several points but doesn't link them. |
| **Relate** | Relational | Explains how ideas connect to each other. Writing shows cause-effect, comparison, or integration. |
| **Transfer** | Extended Abstract | Applies ideas to new contexts. Writing uses knowledge from one domain to illuminate another. |

**Observable signals:**

| Signal Source | Surface Indicator | Deep Indicator |
|--------------|-------------------|----------------|
| Maze | Correct on literal items, wrong on inferential | Correct on both literal and inferential |
| AWE Evidence | Quotes text without explanation | Explains *why* evidence supports the claim |
| Writing content | Restates what was read | Connects reading to other texts, personal experience, or new questions |
| Cross-essay | Same ideas reappear unchanged | Ideas evolve across essays — student refines or challenges prior thinking |

---

### Dimension 2: Approach to Structure

*Adapted from Sternberg's Forms (monarchic → anarchic) + Legislative/Executive distinction*

How does this student organize their thinking?

```
Systematic ◄────────────────────────────► Exploratory
```

| Style | What It Looks Like |
|-------|-------------------|
| **Systematic** | Clear thesis → organized evidence → logical conclusion. Follows conventions. Writing has predictable structure. Revisions focus on precision. |
| **Methodical** | Has a plan but adapts it. Tries structures taught in class. Some flexibility within a framework. |
| **Intuitive** | Structure emerges during writing rather than before. Ideas may be strong but organization is loose. May jump between ideas. |
| **Exploratory** | Follows curiosity. Writing may be nonlinear, surprising, creative. Structure is unconventional. May produce novel insights OR unfocused rambling. |

**Observable signals:**

| Signal Source | Systematic Indicator | Exploratory Indicator |
|--------------|---------------------|----------------------|
| Writing structure | Consistent paragraph count and organization across essays | Variable structure, adapts to content |
| Connective words | "First, Second, In conclusion" pattern | Varied transitions, or stream-of-consciousness flow |
| Revision behavior | Edits for clarity and precision within existing structure | Restructures paragraphs, moves/deletes whole sections |
| Compositional Range | Narrow but polished | Wide but sometimes uneven |
| ORF reading style | Steady pace, few errors | Variable pace — speeds up on engaging parts, slows on difficult parts |

---

### Dimension 3: Orientation to Evidence

*Adapted from Bloom's Analyze/Evaluate levels + Perry's dualism/multiplicity/relativism*

How does this student handle evidence and multiple perspectives?

```
Convergent ◄────────────────────────────► Divergent
```

| Style | What It Looks Like |
|-------|-------------------|
| **Convergent** | Selects one strong piece of evidence. Builds a tight argument. Prefers certainty. Sees issues as having right answers. |
| **Focused** | Uses a few pieces of evidence, prioritizes the strongest. Acknowledges complexity but drives toward a conclusion. |
| **Balanced** | Presents multiple viewpoints fairly. May struggle to commit to a position. Sees merit in different perspectives. |
| **Divergent** | Explores many angles. Raises new questions. Comfortable with ambiguity. May have difficulty reaching closure. |

**Observable signals:**

| Signal Source | Convergent Indicator | Divergent Indicator |
|--------------|---------------------|---------------------|
| AWE Evidence | One or two strong quotes, well-explained | Multiple sources referenced, less depth per source |
| Writing arguments | Strong claim, may dismiss counterarguments | Weaker claim but acknowledges competing views |
| Maze behavior | Fast, decisive selections | Slower, may change answers (if tracked) |
| Cross-unit writing | Consistent position on similar topics | Position shifts as student encounters new information |
| Response to ambiguous text | Picks one interpretation | Holds multiple interpretations simultaneously |

## Part 2: The Thinking Types

The three dimensions combine into profiles. Rather than labeling students with a fixed type, the system describes their **current thinking mode** — which can change across tasks, units, and time.

### Primary Types

| Type | Depth | Structure | Evidence | Description |
|------|-------|-----------|----------|-------------|
| **The Architect** | Deep | Systematic | Convergent | Builds carefully reasoned arguments with precise structure. Strength: rigor. Growth area: flexibility and creative risk. |
| **The Explorer** | Deep | Exploratory | Divergent | Pursues ideas across boundaries, makes unexpected connections. Strength: originality. Growth area: organization and follow-through. |
| **The Detective** | Deep | Systematic | Divergent | Methodically examines multiple angles before reaching a conclusion. Strength: thoroughness. Growth area: efficiency and decisiveness. |
| **The Advocate** | Deep | Exploratory | Convergent | Passionate about a position, argues with creative energy. Strength: persuasion. Growth area: considering opposing views. |
| **The Reporter** | Surface | Systematic | Convergent | Accurately captures and organizes information. Strength: reliability. Growth area: moving beyond reporting to analysis. |
| **The Collector** | Surface | Systematic | Divergent | Gathers lots of information from many sources but stays at the surface. Strength: breadth. Growth area: depth and synthesis. |
| **The Improviser** | Surface | Exploratory | Convergent | Quick responses based on first instincts. Strength: speed and confidence. Growth area: slowing down to deepen thinking. |
| **The Wanderer** | Surface | Exploratory | Divergent | Touches many ideas without developing any. Strength: curiosity. Growth area: focus and commitment to an idea. |

### How Types Map to Cognitive Capacities

| Type | Typically Strong Capacities | Typically Developing Capacities |
|------|---------------------------|-------------------------------|
| Architect | Evidence Evaluation, Pattern Recognition, Metacognition | Analogical Thinking, Compositional Range |
| Explorer | Analogical Thinking, Compositional Range, Perspective Taking | Evidence Evaluation, Pattern Recognition |
| Detective | Inferential Reasoning, Perspective Taking, Metacognition | Compositional Range |
| Advocate | Evidence Evaluation, Analogical Thinking | Perspective Taking, Ambiguity Tolerance |
| Reporter | Literal Comprehension, Pattern Recognition | Inferential Reasoning, Analogical Thinking |
| Collector | Literal Comprehension, Perspective Taking | Evidence Evaluation, Metacognition |
| Improviser | Literal Comprehension, Compositional Range | Metacognition, Evidence Evaluation |
| Wanderer | Compositional Range, Perspective Taking | Literal Comprehension, Evidence Evaluation, Metacognition |

## Part 3: Dynamic Profiles — Types Change Over Time

### 3.1 Per-Task Typing

The same student may exhibit different types in different contexts:

```
Maria Chen, Grade 6

Unit 6A (Dahl & Narrative):        Explorer
  → Creative writing about personal experience
  → Made unexpected connections between Dahl's style and her own life
  → Loose structure but original ideas

Unit 6B (Mysteries & Investigations): Detective  
  → Analytical writing about "The Secret of the Yellow Death"
  → Systematically examined evidence from the text
  → Considered multiple hypotheses before concluding

Unit 6D (The Greeks):              Reporter
  → Informational writing about Greek democracy
  → Accurately summarized key facts
  → Did not connect to modern parallels or offer analysis
```

### 3.2 What Type Shifts Tell Us

| Pattern | Interpretation |
|---------|---------------|
| Consistent type across all units | Strong, stable thinking preference — leverage it, but also design exercises to stretch |
| Deep types in creative tasks, Surface in analytical | Student engages deeply when emotionally connected; needs scaffolding for analytical engagement |
| Shift from Surface → Deep over the year | Growth in cognitive capacity — the system is working |
| Shift from Deep → Surface | Possible disengagement, increased difficulty, or external factors — flag for teacher |
| Different types for reading vs writing | Reading comprehension may be ahead of production ability, or vice versa |

### 3.3 Growth Trajectories

Inspired by SOLO Taxonomy's progression, the ideal growth trajectory is:

```
Beginning of Year                              End of Year
     │                                              │
     ▼                                              ▼
  Reporter ──→ Collector ──→ Detective ──→ Architect
  (Surface,    (Surface,     (Deep,        (Deep,
   Systematic,  Systematic,   Systematic,   Systematic,
   Convergent)  Divergent)    Divergent)    Convergent)
     │              │             │              │
     └── Learning ──┘── to go ───┘── deeper ────┘
         to gather      and connect    and commit
         more info      ideas          to a position
```

But this is NOT the only valid trajectory. An equally valuable path:

```
  Improviser ──→ Advocate ──→ Explorer ──→ Detective
  (Surface,      (Deep,       (Deep,       (Deep,
   Exploratory,   Exploratory,  Exploratory,  Systematic,
   Convergent)    Convergent)   Divergent)    Divergent)
     │                │             │              │
     └── Learning ────┘── to go ───┘── to see ────┘
         to slow down    deeper        other sides
         and think                     and organize
```

### 4.3 What Students See

Students NEVER see their type label. Instead, they receive:

| What They See | Example |
|--------------|---------|
| AWE feedback | "Your essay has a clear focus. To strengthen it, try explaining *why* your evidence supports your claim." |
| Reading progress | "You read 155 words per minute with 97% accuracy — great job!" |
| Writing prompts | Scaffolded prompts that subtly target their growth area (the system selects these based on type, but the student doesn't know) |

## Part 5: Detection Logic

### 5.1 How to Determine a Student's Current Type

#### Step 1: Score each dimension (0-10 scale)

**Depth of Processing:**
```
depth_score = weighted_average(
    maze_inferential_accuracy     × 0.15,
    awe_evidence_score            × 0.20,
    reasoning_present_in_writing  × 0.25,   // does the essay have "because/therefore"?
    cross_text_connections        × 0.20,   // references to other readings
    idea_evolution_across_essays  × 0.20    // do ideas develop or stay static?
)
```

**Structure (Systematic ↔ Exploratory):**
```
structure_score = weighted_average(
    structural_consistency        × 0.25,   // cosine sim of essay shapes
    connective_diversity          × 0.20,   // variety of transition words
    revision_type                 × 0.25,   // surface edits vs restructuring
    paragraph_organization        × 0.15,   // clear topic sentences?
    orf_pace_variability          × 0.15    // steady vs variable reading pace
)
// low score = systematic, high score = exploratory
```

**Evidence Orientation (Convergent ↔ Divergent):**
```
evidence_score = weighted_average(
    num_evidence_sources          × 0.25,   // how many distinct pieces cited
    counterargument_present       × 0.25,   // acknowledges other views?
    position_consistency          × 0.20,   // same stance across essays on similar topics?
    maze_answer_changes           × 0.15,   // changes answers (if tracked)
    depth_per_evidence            × 0.15    // deep on few, or shallow on many?
)
// low score = convergent, high score = divergent
```

#### Step 2: Map to type

```
if depth >= 5:
    if structure < 5 and evidence < 5:  → Architect
    if structure < 5 and evidence >= 5: → Detective
    if structure >= 5 and evidence < 5: → Advocate
    if structure >= 5 and evidence >= 5: → Explorer
else:
    if structure < 5 and evidence < 5:  → Reporter
    if structure < 5 and evidence >= 5: → Collector
    if structure >= 5 and evidence < 5: → Improviser
    if structure >= 5 and evidence >= 5: → Wanderer
```

#### Step 3: Confidence check

Only assign a type when there are enough signals:
- Minimum 3 writing samples in the current unit
- At least 1 mCLASS data point in the current school year
- If insufficient data → show "Developing Profile" instead of a type

## Part 6: Database Additions

### thinking_profiles
```sql
CREATE TABLE thinking_profiles (
    profile_id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    student_id          UUID NOT NULL REFERENCES students(student_id),
    unit_id             VARCHAR(20),            -- null for aggregate period profile
    assessment_period   VARCHAR(3),             -- 'BOY', 'MOY', 'EOY' for aggregate
    school_year         VARCHAR(9) NOT NULL,
    depth_score         DECIMAL(4,2) NOT NULL,  -- 0-10
    structure_score     DECIMAL(4,2) NOT NULL,  -- 0-10 (low=systematic, high=exploratory)
    evidence_score      DECIMAL(4,2) NOT NULL,  -- 0-10 (low=convergent, high=divergent)
    thinking_type       VARCHAR(20) NOT NULL,   -- 'architect','explorer','detective', etc.
    confidence          DECIMAL(3,2) NOT NULL,  -- 0-1, based on signal count
    signal_count        INT NOT NULL,           -- how many data points contributed
    evidence_summary    JSONB,                  -- key signals that drove classification
    created_at          TIMESTAMPTZ DEFAULT now(),
    UNIQUE (student_id, unit_id, school_year)
);
```

### thinking_profile_history (track type changes)
```sql
CREATE TABLE thinking_profile_history (
    history_id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    student_id          UUID NOT NULL REFERENCES students(student_id),
    school_year         VARCHAR(9) NOT NULL,
    type_sequence       JSONB NOT NULL,         -- ["explorer","detective","explorer","reporter"]
    units               JSONB NOT NULL,         -- ["6A","6B","6C","6D"]
    depth_trend         VARCHAR(10),            -- 'increasing', 'stable', 'decreasing'
    structure_trend     VARCHAR(10),
    evidence_trend      VARCHAR(10),
    notable_shifts      JSONB,                  -- flagged shifts with context
    updated_at          TIMESTAMPTZ DEFAULT now()
);
```
